# Facebook Signup Page Clone

A pixel-perfect, fully responsive recreation of the classic Facebook Sign Up interface built using semantic HTML5 and custom CSS3.

## 🚀 Features
* **Grid & Flexbox Layouts:** Utilized modern CSS layouts to precisely match the sizing and spacing of the original design.
* **Responsive Styling:** Adaptive styling with media queries ensuring optimal rendering on both desktop browsers and mobile devices.
* **Semantic Code:** Structured with clean and clean HTML elements representing forms, inputs, and structure labels.

## 🛠️ Built With
* **HTML5:** For the page structure and semantic inputs.
* **CSS3:** For complete styling, visual alignment, shadows, hover states, and responsiveness.
